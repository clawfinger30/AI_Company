- `README.md` はまだ古い機能概要を含む。特にフォロー/いいね/リツイート自動化を主要機能として記載しているため、後続で更新推奨。
- `src/services/scheduler.js` と `src/services/xAutomation.js` には旧エンゲージメント自動化が残るが、現行 `main.js` では初期化されていない。
- 起動済みの配布アプリには今回のソース変更は自動反映されない。再ビルド/再起動が必要。
